6.1 简洁高效的rem适配方案flexible.js

手机淘宝团队出的简洁高效移动端适配库 我们再也不需要写不同的屏幕的媒体查询，以为里面js做了处理 他的原来是把当前设备划分10等份，但是不同设备下，比例还是一致的。 我们要做的就是确定好我们当前设备的html文字大小就可以了 比如当前设计稿是750px，那么我们只需要把html文字大小设置为75px(750px / 10) 就可以， 剩余的，让flexible.js来去算 github地址:https://github.com/amfe/lib-flexible/blob/master/index.html
