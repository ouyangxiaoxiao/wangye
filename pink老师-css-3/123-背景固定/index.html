3.css的背景

通过css背景属性，可以给页面元素添加背景样式

背景属性可以设置背景颜色，背景图片，背景平铺，背景图片位置，背景图像固定等

3.5 背景图像固定(背景附着)

background-attachment属性设置背景图像是否固定或者随着页面的其余部分滚动

background-attachment 后期可以制作视差滚动效果

background-attachment: scroll|fixed

参数 作用
scroll 背景图像是随对象内容滚动的
fixed 背景图像固定
