5.3css属性书写顺序(重点)

生活中衡量一个人有气质

穿着打扮 举止言行 等等

编程中如何衡量一个人的代码能力？

规范标准 优雅高质量等等 一个词形容 专业 从代码中看出来是否有经验

建议遵循以下顺序:
1.布局定位属性:display样式/position定位/float浮动/clear清除/visibility可见性/overflow溢出隐藏(建议display第一个写，毕竟关系到模式)
2.自身属性:width宽度/height高度/margin外边距/padding内边距/background背景
3.文本属性:color颜色/font字体/text-decoration文本装饰/text-align水平居中/vertical-align垂直居中/white-space是否换行/break-word断点换行
4.其它属性(css3):content居中/光标类型/border边框/border-radius边框圆角/box-shadow盒子阴影/text-shadow文字阴影/background:linear-gradient背景渐变
