2.css3的新特性

css新增选择器

css3给我们新增了选择器，可以更加便捷，更加自由的选择目标元素。 1.属性选择器 2.结构伪类选择器 3.伪元素选择器 2.4伪元素选择器(重点) 伪元素选择器可以帮助我们利用css创建新的标签元素，而不需要HTML标签，从而简化HTML结构。

选择符 简介
::before 在元素内部插入内容
::after 在元素内部后面插入内容

注意：

  • before和after创建一个元素，但是属于行内元素
  • 新创建的元素在文档中是找不到的，所以我们称为伪元素
  • 语法：element::before{}
  • before和after必须有content属性
  • before在父元素前面创建元素，after在父元素内容后面插入元素
  • 伪元素和标签选择器一样，权重都为1
