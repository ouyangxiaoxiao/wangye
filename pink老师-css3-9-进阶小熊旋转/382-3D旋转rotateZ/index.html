5.动画

动画(animation)是css3中最有颠覆性的特征之一，可通过精确节点来控制一个或一组动画，常用来实现复杂的动画效果 相比过度，动画啊可以实现更多变化，更多控制，连续自动播放等效果

5.1动画的基本使用

制作动画分为两步 1.先定义动画 2.在使用(调用)动画

1.用keyframes定义动画(类似定义选择器)

@keyframes 动画名称 { /*开始状态*/ 0% { transform: translateX(0px); } /*结束状态*/ 100% { transform: translateX(1000px); } }

动画序列

0%是动画开始，100%是动画完成，这样的规则就是动画序列。 在@keyframes中规定某项css样式，就能创建由当前样式逐渐改为新样式的动画效果。 动画是使元素从一种样式逐渐变化为另外一种样式的效果。您可以改变任意多的样式任意多的次数。 请用百分比来规定变化发生的时间或用关键词 form和to 等同于0%和100%

6. 3D转换

我们生活的环境是3D 的，照片就是3D物体在2D平面呈现的例子。

有什么特点

近大远小 物体后面遮挡不可见 当我们在网页上构建3D 效果的时候参考这些特点就能产出3D 效果。

6.1 三维系

三维坐标系就是指立体空间，立体空间是由3个轴共同组成的。 x轴:水平向右 注意：右边是正值，左边是负值 y轴：垂直向下 注意：y下面是正值，上面是负值 z轴：垂直屏幕 注意：往外面是正值，往后面是负值 3D转换 我们主要学习工作中 常用的3D位移和3D转换、

主要知识点：

3D位移:translate3d(x,y,z)-穿丝利特3d 3D旋转:rotate3d(x,y,z)-弱贴的3d 透视:perspective-珀斯派台忒无 3D呈现transfrom-style-穿丝纲目-思黛欧

6.2 3D 移动 translate3d

3D移动在2D移动的基础上多加了一个可以移动的方向，就是 z轴方向。 translform:translateX(100px):仅仅在X轴上移动 translform:translateY(100px):仅仅在Y轴上移动 translform:translateZ(100px):仅仅在Z轴上移动(注意translateZ一般都是用单位px) translform:translate3d(x,y,z):其中x，y，z 分别指要移动的轴的方向的距离

透视 perspective

在2D平面产生远大近小视觉立体，但是只是效果二维的 如果想要在网页产生3D效果需要透视(理解3D物体投影在2D平面内) 模拟人类的视觉位置，可以安排一只眼去看 透视我们也称为视距：视距就是人的眼睛到屏幕的距离 距离视觉点越近的在电脑平面中成像越大，越远越小 透视的单位是像素

透视写在被观察元素父盒子上面 d:就是视距，视距就是一个距离人的眼睛到屏幕的距离 z:就是z轴，物体距离屏幕的距离，z轴越大(正值)我们看到的物体就越大

6.5 3D 旋转 rotate3d

3D 旋转指可以让元素在三维平面内沿着x轴，y轴，z轴或者自定的轴进行旋转。 语法 tansform:rotateX(45deg):沿着x轴正方向旋转45度-单杠 tansform:rotateY(45deg):沿着y轴正方向旋转45度-钢管舞 tansform:rotateZ(45deg):沿着z轴正方向旋转45度-抽奖转盘 tansform:rotate3d(x,y,z,deg):沿着自定义轴旋转deg为角度(了解即可) xyz 是表示旋转轴的矢量，是标示你是否希望沿着该轴旋转，最后一个标示旋转的角度 transform:rotate3d(1,0,0,45deg)就是沿着x轴旋转45deg transform:rotate3d(0,1,0,45deg)就是沿着y轴旋转45deg 对于元素旋转的方向的判断 我们需要学习一个左手准则。 左手准则 左手的手指指向x轴的方向 其余手指的完全方向就是该元素沿着x轴旋转的方向 左手的手指指向Y轴正方向 其余手指的弯曲方向就是该元素沿着y轴旋转的方向(正值)

6.6 3D呈现 transform-style
